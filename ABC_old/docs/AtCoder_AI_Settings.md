# AtCoder AI対応設定 - 設定内容の説明

AtCoderのAI支援コーディング禁止ルールに対応するため、以下の機能強化を実装しました。

## 🚀 実装された機能

### 1. 予測変換・IntelliSense強化
- **即座の候補表示**: `quickSuggestionsDelay: 0` で遅延なし
- **積極的な単語ベース補完**: 全ドキュメントから候補を収集
- **スニペット優先**: コードスニペットが最優先で表示
- **パラメータヒント**: 関数の引数情報を自動表示

### 2. 豊富なスニペット集
#### C++用スニペット (70+種類)
- **基本テンプレート**: `main`, `template`, `basic`
- **入力系**: `intn`, `inta`, `int2d`, `instr`
- **ループ系**: `rep`, `rep1`, `for`
- **アルゴリズム**: `sort`, `lb`, `ub`, `bfs`, `dfs`
- **データ構造**: `graph`, `uf`, `pq`, `map`, `set`
- **数学**: `gcd`, `lcm`, `mod`, `powmod`, `fact`

#### Python用スニペット (60+種類)
- **基本テンプレート**: `main`, `template`, `basic`
- **入力系**: `intn`, `inta`, `int2d`, `instr`
- **ループ系**: `for`, `enum`, `range`
- **アルゴリズム**: `sort`, `bisect`, `bfs`, `dfs`
- **データ構造**: `graph`, `uf`, `heap`, `counter`
- **関数型**: `listcomp`, `map`, `filter`

### 3. 強化されたテンプレート
#### C++テンプレート
- **包括的な型定義**: 30+の便利な型エイリアス
- **完全なマクロセット**: 15+の頻出マクロ
- **数学・アルゴリズム関数**: 20+の実装済み関数
- **デバッグ支援**: ローカル環境用デバッグマクロ

#### Pythonテンプレート
- **効率的な入出力**: 最適化された入出力ヘルパー
- **数学・アルゴリズム**: 素数、約数、座標圧縮など
- **データ構造**: Union-Find、BIT、セグメントツリー
- **幾何学**: マンハッタン距離、チェビシェフ距離

### 4. 便利なキーバインド
- **Ctrl+Shift+T**: テンプレート挿入
- **Ctrl+Shift+I**: 数値入力スニペット
- **Ctrl+Shift+A**: 配列入力スニペット
- **Ctrl+Shift+F**: forループスニペット
- **Ctrl+Shift+Y**: Yes/No出力
- **F5**: ビルド・実行

### 5. 追加ヘッダーファイル
- **atcoder.hpp**: C++用包括的ライブラリ
- **よく使う関数をすべて収録**: Union-Find、BIT、セグ木、二分探索、素数など
- **テンプレートとセットで使用可能**

## 📋 使用方法

### スニペットの使用
1. エディタでキーワードを入力（例: `main`, `rep`, `bfs`）
2. Ctrl+Spaceで候補表示
3. Enterで挿入

### テンプレートの使用
1. 新しいファイルを開く
2. Ctrl+Shift+Tでテンプレート挿入
3. 必要な部分を編集

### ビルドと実行
1. C++: F5キーでビルド・実行
2. Python: F5キーで実行
3. エラーがある場合は下部パネルに表示

## 🔧 トラブルシューティング

### スニペットが表示されない場合
1. Ctrl+Spaceで手動表示
2. 言語モードが正しく設定されているか確認
3. 設定の`editor.suggest.showSnippets`が有効か確認

### ビルドエラーの場合
1. C++コンパイラー（g++）がインストールされているか確認
2. パスが通っているか確認
3. ファイルが正しく保存されているか確認

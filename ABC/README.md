# AtCoder ABC フォルダテンプレート作成ツール v2.0

## 🌟 概要

AtCoder Beginner Contest (ABC) 参加用のフォルダを自動生成する高機能PowerShellスクリプトです。

## ✨ 新機能・改善点

### v2.0の新機能
- 🎨 **視覚的なUI**: 絵文字とカラフルな出力で使いやすさ向上
- ⚙️ **柔軟な言語選択**: C++のみ、Pythonのみ、または両方を選択可能
- 📅 **コンテスト日付設定**: 開催日を指定可能
- 🔧 **VS Code統合**: タスクランナー設定を自動生成
- 🧪 **テストファイル**: 各問題用の入力ファイルテンプレート
- 💪 **強制上書き**: 既存フォルダの上書きオプション
- 📂 **自動オープン**: 作成後にフォルダやVS Codeを自動で開く
- 🚀 **簡単実行**: バッチファイル（.bat）による簡単実行

### 主な改善点
- **エラーハンドリング強化**: 詳細なエラーメッセージと前提条件チェック
- **パラメータバリデーション**: 不正な入力値の検証
- **ヘルプ機能**: 詳細な使用方法説明
- **進捗表示**: 処理状況の視覚的フィードバック

## 🚀 使用方法

### 基本的な使い方

```powershell
# 最も簡単な方法（バッチファイル使用）
abc.bat 372

# PowerShellスクリプト直接実行
.\create_abc.ps1 372
```

### 高度なオプション

```powershell
# すべてのオプションを使用
.\create_abc.ps1 372 -Force -Open -VSCode -CreateTestFiles -Language both -Date "2025-01-15"

# C++のみでフォルダ作成
.\create_abc.ps1 372 -Language cpp

# 既存フォルダを強制上書き
.\create_abc.ps1 372 -Force

# 作成後にVS Codeで開く
.\create_abc.ps1 372 -VSCode
```

## 📋 パラメータ説明

| パラメータ | 説明 | 例 |
|------------|------|-----|
| `ContestNumber` | コンテスト回数（必須） | `372` |
| `-Force` | 既存フォルダを強制上書き | `-Force` |
| `-Open` | エクスプローラーでフォルダを開く | `-Open` |
| `-VSCode` | VS Codeでフォルダを開く | `-VSCode` |
| `-CreateTestFiles` | テスト用入力ファイルを作成 | `-CreateTestFiles` |
| `-Language` | 使用言語 (cpp/python/both) | `-Language cpp` |
| `-Date` | 開催日 (YYYY-MM-DD) | `-Date "2025-01-15"` |

## 🔧 カスタマイズ

### テンプレートのカスタマイズ
`template/` フォルダ内のファイルを編集することで、生成されるテンプレートをカスタマイズできます。

### 設定ファイル
`config.ps1` を編集してデフォルトの動作を変更できます。

## 🚨 注意事項

- PowerShell 5.1以上が必要
- 同じ回数のフォルダが既に存在する場合は `-Force` オプションが必要
- テンプレートフォルダ (`template/`) が必要
- VS Code機能を使用する場合は VS Code がインストールされている必要があります

## 🔧 トラブルシューティング

### 実行ポリシーエラー
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### テンプレートファイルが見つからない
`template/` フォルダに必要なファイルがあることを確認してください。

### VS Codeが開かない
VS Codeがシステムの PATH に追加されていることを確認してください。
